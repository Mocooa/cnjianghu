# Aesthetic China Second-Layer Tree v1

Archived working spec for the currently expanded `Aesthetic China` second-layer content tree.

Principles used in this version:
- Write for foreigners trying to understand China, not for inward-facing cultural description.
- Use content-form buckets rather than academic sub-taxonomies.
- Keep the tree easy to review in plain text: one node per line.
- Allow uneven branch density where the subject naturally has more topic surface.
- Treat this as the current expanded subset of the pillar, not the final exhaustive tree.
