# Top 30 Launch Slate v1

Cross-pillar launch slate extracted from the six tagged title-bank files.

This file is not a new title-ideation pass.
It is a selection layer built only from titles that already exist in the current production banks.

## Selection Rules

- Source pool:
  - Only titles already present in the six `*-title-bank-v1.md` files
- First filter:
  - Prioritize titles already tagged `Launch first`
- Second filter:
  - Favor titles also tagged `High shareability`
- Balance rule:
  - Keep representation from all six pillars
  - Cap the slate at `5` titles per pillar
- Format rule:
  - Use lighter titles for the front of the launch
  - Use deeper authority pieces in the back half
- Support rule:
  - Treat glossary-heavy concepts as dependencies when needed
  - Do not overload the first launch slate with glossary-led front-door pieces

## Format Mix

- `Quick Bites`: `19`
- `Misread / Actually`: `5`
- `Deep Dives`: `6`
- `Scene Decoder`: `0`
- `Glossary-led`: `0`

Reason:
- The first launch slate should optimize for entry, not completeness.
- `Scene Decoder` and `Glossary-led` remain support layers for later sequencing, not the first front door.
