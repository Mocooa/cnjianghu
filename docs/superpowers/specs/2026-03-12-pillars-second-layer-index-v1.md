# Six-Pillar Second-Layer Index v1

Master index for the current archived second-layer content trees across all six pillars.

This file exists to make review easier:
- one place to see what has already been expanded

## Working Rules

- Audience lens: foreigners trying to understand China
- Structure model: `dossier -> content-form buckets -> topic directions`
- Content-form buckets:
  - `Quick Bites`
  - `Artifact Cards`
  - `Glossary Terms`
  - `Scene Decoder`
  - `Misread / Actually`
  - `Deep Dives`
  - `Relays`
  - `Trails`
- Review format: one node per line
- Current state: all six pillars are archived as `partial v1`, not final exhaustive trees
